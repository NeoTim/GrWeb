通用棋牌遊戲平台

1.通用版棋牌遊戲簡介

     通用版棋牌遊戲使用VC++2003.NET開發，數據庫可使用MSsqlServer或ACCESS。具有優越的數據處理功能；編程遵循匈牙利命名規範；服務器兼容Windows 系列版本，採用IOCP完成端口，令服務器網絡資源的發揮到極致，軟件兼容性極強，遊戲畫面細膩而美觀，操作性能優越。 遊戲規則與聯眾、中國遊戲中心幾乎一致，己獲得國家版權局頒發的計算機軟件著作權登記。

2. 通用版棋牌遊戲內容

      包括:四國軍棋、三打一、炒地皮、紅心大戰、拱豬、象棋、圍棋、斗地主、玉環80分、510K、玉環打通、嵊州105、憋七、跑得快、雙扣、原子、梭哈、鋤大地、益友麻將、大慶麻將、溫嶺16張麻將、等遊戲

3. 通用版棋牌遊戲特點

1.通用版棋牌遊戲分為登陸服務器、遊戲服務器、中央服務器以及相應的數據庫服務器， 客戶端大廳、遊戲客戶端DLL構成。
2.通用版棋牌遊戲大廳服務器採用TCP短連接模式，單台服務器可承載5000以上用戶同時使用。
3.通用版棋牌遊戲服務器採用TCP長連接，單台遊戲服務器可同時支持5000人在線遊戲，如進行某些優化可實現更多用戶支持。
4.通用版棋牌遊戲服務器單獨記錄遊戲用戶積分狀態，並採用完全可分佈、可維護的擺放方案。可動態擴展服務器，定制遊戲房間、遊戲桌。 服務器充分利用NT系統能力，底層結構清晰，並可方便的開發新遊戲、新遊戲服務器。

4. 通用版棋牌遊戲優勢

      客戶端採用GDI+模式工作，對於棋牌遊戲可充分支持，如需進行複雜圖像操作，可套用DX\DDRAW進行擴展，實現靚麗的遊戲表現。 客戶端開發模式簡單，接口清晰。對於一般類型的棋牌遊戲，可在1週內實現一款。 服務器代碼實現所有的網絡、數據庫、棋牌桌管理的功能。 客戶端包含一個用於開發遊戲所需要的DLL框架工程。即使生手也可以在短時間內進入開發狀態。

5. 通用版棋牌遊戲特性

安全性：
     數據庫部分敏感信息加密、客戶端IPC通信和網絡通信都採用自主開發的加密技術、客戶端用戶密碼加密等，同時，對網絡包採用自我的加密算法加密，本地保存的敏感信息採用多層加密，有可逆的與不可逆的，數據庫部分的敏感信息採用MD5算法加密，還利用端口變換，服務器部署環境增加安全和保密；保證不會由於系統、遊戲漏洞造成遊戲金幣流失，外掛等狀況。 高壓性： 單台服務器最高並發可支持5000人同時在線。人數超過5000時，可進行多台服務器分佈式處理，網絡、CPU佔用率極低，加強了平台對服務器要求的伸縮性。
* 1. 每遊戲服務器的安全並髮用戶人數達到4000人以上。並髮用戶人數的上限達到5000人以上。服務器配置為XEON 2.4G ×2、1024M×2內存、73G硬盤。
* 2. 當平台人數達到4000人時，在帶寬512K的ADSL的條件下，用戶登錄、進入遊戲房間等的平均時間不高於20秒。
* 3. 每聊天室的安全並髮用戶數達到500人。
* 4. 當人數達到500人時，聊天信息從發送到顯示的時間不高於5秒。網絡環境為512K ADSL。
* 5. 正常維護頻率不高於2個月維護一次，不需要停機維護，每次維護在半小時以內。
