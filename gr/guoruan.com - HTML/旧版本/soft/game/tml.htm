■WebP2P视频休闲棋牌游戏

　　　WebP2P视频休闲棋牌游戏并非传统棋牌游戏的升级产品，两者采用相同的TMLib技术开发，在架构上采用更简便的Web扁平架构，主要增强游戏的休闲性和易玩性，属于同一技术的两种表现形式。



※功能简介
　　WebP2P（基于WEB扩展的采用TMLib类库开发的基于网格和SOA架构的视频语音P2P人工智能游戏系统）是采用TMLib技术开发的新型Web游戏，包含了前五代所有的功能，不但支持P2P、网格和SOA架构、人工智能机器人、视频语音，而且增加了基于Web的可扩展性，运营部署更简便，技术要求更低，自定义性更强。

   

※产品优势
　　现在相对各种专用客户端的网络游戏来说，Web服务拥有更大量稳定的用户群。因此我们将网络棋牌游戏定位为是对Web功能的延伸和完善，提供一种简单轻松的休闲娱乐环境，不需要复杂的游戏系统。而现在对用户的这方面需求的服务还是十分的有限。因此，这应该是网络上的一个前景广阔的未来市场。

　　在我们构想的网络社区里，将有轻松舒适的环境，友好的用户界面，个性化的选择，便捷的交流方式，辅以好玩时尚的休闲小游戏，加上特殊的游戏模式和任务等游戏元素，一定可以吸引更多的网络用户。

　　另一方面，由于我们的游戏使用IE浏览器WEB方式来运行，对于大部分网吧上网用户来说，使用我们的游戏和浏览一般网站一样方便；而且，这样的方式，对大多数用户来说，会更符合他们的消费习惯。

　　运营方面，游戏采用WEB架构，大部分界面采用Web实现，对于运营商来说，可以随时根据自己的需要调整界面和修改功能，懂开发网站的人就可以开发游戏，降低了运营门槛，适合运营行业的未来趋势。
   

※游戏场景


万里长城

　　游戏支持动态税率设置吗？抽水可以固定吗？负分可以抽水吗？游戏支持电脑分配座位吗？游戏支持擂台赛设置吗？这是众多玩家反复问我们的问题。对本游戏平台来说，这些并不是新问题。
　　 本平台的策划设计都放在场景和任务上。比如说：哪种游戏配合什么背景故事，哪种场景配合什么任务，设计什么任务更吸引玩家游戏，四人斗地主游戏跟长城有什么关系，中国象棋跟太阳神有什么关系……等等……

   

　　新游戏的这些设计融入了RPG游戏的特点。疯子创作室将此作为主要的扩展要素。疯子创作室设计风格是难以置信的强调扩展性。每一代在开发时都会融入下一代的设计理念。虽然当时没有足够的技术实现这一设计，但疯子创作室都会预留大量铺垫。一旦疯子创作室掌握实现这种理念的足够的技术，便会将所有技术整合成一个新的平台，推出下一代的产品。通常的换代开发周期是两年，第一年用来整合所有技术，第二年才开始开发新一代的平台。而在开发新一代的平台的时候，又会融入下一代的设计理念。在开发EUQQ XP 第5代（TMLib类库架构）的时候，疯子创作室做了大量工作能使TMLib类库架构平移到Web平台上，为此花费整整一年的时间将MFC\GDI+\DX剥离出来，这些技术将导致平台无法Web化，平台开发的时候，很多功能不是说技术上解决不了。而是综合考虑功能、性能、安全等各方面的平衡性，才最后决定怎么做的。一半时间在找平衡。在2007年6月才做出了第一个能在Web浏览器上运行的TMLib类库Runtime Environment，这就是本游戏平台的开发基础。而这种设计为平滑演变为大型MMORPG平台已经做了大量准备工作。
   

狮身疯子像

　　未来的应用是技术整合的应用。本游戏平台是一个整合了众多新技术的平台，而整合这些技术足足花了我们两年的时间，比预计整整拖后了一年。真正的开发是从第三年才开始。但跟为MMORPG比起来，仍然缺乏很多基础要素。在游戏策划时，还是保守的保留了大量传统的棋牌游戏的特点，砍去了大部分的创造性改进。不过，游戏中还是设计一种独立的执行线程来实现类似MMORPG中的任务系统。其实对这种任务系统存在的意思，还在博弈中。是否能获得玩家和市场的认可还是未知数。只有到系统真正上线运营了，才能最后确定这种功能将以何种形态存在。

雪战到底

　　这个创意来源一段真实的童年的回忆。在雪地里玩猜扑克，猜错了就要挨雪球，挨到认输为止。经过我们精心策划，在扑克麻将里都加入了雪战系统。而游戏中的雪球是跟金币、魅力挂钩的。比如，
任务型： 游戏每局随机组合成一个牌型，当你成功的打出这个牌型，将有一个雪球砸向对方，对方金币和魅力都会锐减。
助威型：比如斗地主打出了炸弹， 将有一个雪球砸向对方，对方金币和魅力都会锐减。
金币型：直接购买雪球砸向对方。
　　这个系统在正式上线前还要经过长期大量的线上对战测试，确保不出什么错误，这个过程可能可能还要半年以上的时间。 我们希望将自身积累的各种经验整合到游戏中，在合适的时间、合适的情况下推出合适的游戏，那些童年的伙伴看到网上打牌也能象小时候那样打雪战时，估计会激动得厥倒……

宙斯赌场

　　这里百家乐、梭哈、扎金花，会集中在宇宙之神宙斯开的赌场里。当三个游戏输的钱的总和达到一定数额时，宙斯会将这些钱随机送给当前在线游戏的某个玩家。或者你只玩了一局，就中了500万大奖。
　　这是EUQQ Vista™ 第6代的中奖系统，这个系统能够跨游戏、跨服务器、跨网运行。为此我们专门为每个游戏增加了一个通信服务。游戏不再是孤岛。同类游戏可以组成一个赌场、或一个娱乐场……

太阳神像
