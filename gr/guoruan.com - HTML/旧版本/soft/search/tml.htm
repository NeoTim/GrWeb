搜索引擎系统

　
 1.平台简介:
    天盟搜索原名NYS搜索,开发的初衷是协助我们搜集互联网上的信息。经过四年的发展，已经成为一套完善的搜索引擎系统，而且是互联网上第一套公开所有程序（包含部分JAVA代码）的搜索引擎，它使网管们可以轻松搭建真正属于自己的搜索引擎。NYS小组早在 1998 年就开始搜索引擎研发，现已成功推出基于 J2EE+Oracle 和ISAPI+Oracle两个版本的成熟的搜索引擎系统，几乎可适于所有平台。
后台程序运行在网页采集机器上， 24 小时对互联网网页进行超链接分析，将数据提取到 oracle 数据库中，采集速度 Oracle 每天每台网页采集机器增加 6G 数据。
主服务器的前台利用 JSP+Oracle 实现基于J2EE的速度最快的搜索网页。
本系统在 Linux 和 winNT 测试通过，搜索速度远高于在目前百度和一搜的PHP的搜索，现在转让整套系统，同时也寻找项目合作伙伴和投资商，预算需要资金支持 200 万，主要用于构件高速服务器组，我们坚信凭借我们不懈努力，一定会成为最大的中文搜索！
本搜索引擎已经在曲阜师范大学校级项目 NOAS 等众多项目中成功应用。
