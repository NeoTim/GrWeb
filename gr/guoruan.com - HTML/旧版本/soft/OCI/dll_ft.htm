基於OCI開發的ORACLE獨立ADO驅動集成超級管理器—告別SQL*Net的真正高速直連

 您現在所在的位置:Oracle++ DLL驅動(包含Oracle Provider for OLE DB)
—使VisulFoxPro、FLASH、Authorware、Director高速直連Oracle
 1.下載試用
本驅動是基於OCI封裝的一組Oracle函數的動態庫。使開發者可以在VisulFoxPro、FLASH、Authorware、Director等 不支持Oracle存儲過程的開發工具中，直接開發高速穩定的Oracle應用程序。免費測試版下載地址：http://tml.cn/soft/OCI/dll.rar（壓縮包中包含了一套VisulFoxPro連接Oracle源代碼，和一套HTML連接Oracle的源代碼）
 2.VisulFoxPro開發Oracle應用程序說明
Oracle與Visual FoxPro的數據引擎是毫無關係的系統，Visual FoxPro對Oracle的支持只是停留在較低的水平。 Oracle的Recordset無法與Visual FoxPro的內置控件捆綁。其原因是：Visual FoxPro 不能識別Oracle的Recordset，Visual FoxPro只能通過COM的方式識別Oracle的Recordset，不能像表格那樣讀寫它，只能將記錄逐條讀取，而不是一下子認得整個Recordset。這樣Recordset無法與Visual FoxPro的內置控件捆綁，同時也不能對Oracle直接使用Visual FoxPro強大的數據處理功能。
Oracle DLL驅動可以使Visual FoxPro高效讀寫，在Oracle6i以上版本和Visual FoxPro6以下版本中測試通過。在我們提供的測試下載中包含一套VisulFoxPro連接Oracle源代碼。調用方法： //格式為：d=TML.conn("ORACLE服務器IP,ORACLE服務器端口,ORACLE服務名,ORACLE用戶名,ORACLE用戶密碼");
 3.開發說明
使用本驅動開發Ｏｒａｃｌｅ程序的步驟：
1.安裝本驅動
2.在程序中創建對象
如:在FOX中使用：TML=CREATEOBJECT("TML.TMLi")
在JAVA中使用：TML=new ActiveXObject("TML.TMLi");
在VB中使用：set TML=CreateObject("TML.TMLi")
在Delphi中使用：users ComObj var TML:Variant; TML:=CreateOleObject('TML.TMLi');
這裡創建對象名為TML.TMLi,其中庫名為TML,其中TMLi為對象該DLL中封裝了我們獨立開發的ORACLE驅動，以及SQL解釋器,這是一組函數庫，可以在JAVA裡調用，也可以在FOXpro等，支持DLL函數庫的都可以用。這句話是首先要創建函數庫，這個函數庫中對象名就是TML.TMLi，這個是固定的，在FOX也是這樣，就像ZIP
FOX操作Oracle時，不需要AOD，也不需要ODBC，等，不需要其他任何東西，只需要安裝OCIdriverDLL.exe就行了。
3.連接打開
Oracle FOX和VB中使用d=TML.conn("ORACLE_HOST_IP,ORACLE_PORT,ORACLE_SID,ORACLE_User,ORACLE_Password")
JVAA中使用：d=TML.conn("ORACLE_HOST_IP,ORACLE_PORT,ORACLE_SID,ORACLE_User,ORACLE_Password");
上面的d為返回的字符，會提示是否連接成功。 .conn是這個函數庫的一個方法該方法用於設置oracle的連接參數並連接Oracle。
參數是ORACLE_HOST_IP,ORACLE_PORT,ORACLE_SID,ORACLE_User,ORACLE_Password
分別是ORACLE服務器IP，服務器端口，數據庫ID（如LOCALDB），數據庫的用戶名，和密碼。
中間用,隔開。在第一次打開ORACLE服務器只前，首先要調用一次，
如果程序中，需要更改ORACLE服務器，換個用戶或換個數據庫，再要調用一次，更新連接，
如果程序一直用同一個庫，同一個用戶，那不需要再調用。 .conn會返回一個字符串，如果連接成功就返回連接信息，如果連接失敗就連接失敗的錯誤代碼。
4.操作Oracle（Insert,Delete,CREATE TABLE,creat user,creat index，creat function，creat ……等）
FOX和VB中使用：d=TML.ExeSql("insert into T(TEST) values('FOX操作ORACLE BY FOXPRO')")&&;//插入一條記錄
JAVA或JS中使用：d=TML.ExeSql("insert into T(TEST) values('JavaScript操作ORACLE')");//插入一條記錄
上面的d為返回的字符，會提示操作的信息。 .ExeSql是執行SQL語句的方法，參數是可以是任意DML語句（如insert,delete,update），也可以是任意DDL語句
如（creat table,creat user,creat index，creat function，creat ……等，）也可以是Drop等。
Oracle所有的SQL語句，都支持，但不返回數據集。但返回一個結果，如update語句更新的數據庫記錄數。
上面的語句"insert 就是向T表中的插入一條記錄
T表定義如下：CREATE TABLE "T" ("ID" NUMBER(10) NULL,"TEST" VARCHAR2(50) NOT NULL)
可以用 TML.ExeSql("CREATE TABLE T (ID NUMBER(10) NULL,TEST VARCHAR2(50) NOT NULL)")來創建這個表。 5:查詢Oracle（select）
FOX和VB中使用：s=TML.SelSql("Select id,TEST from T where test like '%%' order by id desc",25,document.cookie)
JAVA 或JS中使用：s=TML.SelSql("Select id,TEST from T where test like '%%' order by id desc",25,document.cookie);
SelSql是查詢的方法，專門用來執行SELECT語句。輸入三個參數的格式：SQL語句必須是SELECT,每頁行數,頁數
返回為一個S的字符串，S是一個二維數組，需要自己還原成一個表格，，返回的字符串用“||”來分割每條記錄，用“|”來分割字段
也就是說兩個“||”間的是一條記錄，兩個“|”間是一個字段，字段順序就按SELECT中的順序來排列的，所以SELECT中必須指明字段名，不能用*，但可以用MAX，SUM
//查詢數據庫的語句，格式：SQL,頁行數,頁數。
