Oracle原生驱动

　国软自主开发的数据库驱动程序，能替代ORACLE原有的数据库驱动，有效提高访问速度和执行效率。为应用程序提供独立连接Oracle的驱动，完全不需要SQL*Net，即装即连， 而现存的诸多驱动比如JDBC、ODBC、DOA（Direct Oracle Access）、ODAC（Oracle Data Access Components）都要依赖SQL*Net。 本驱动是从事ORACLE开发的程序员首选组件，特别适合初学者。

Oracle通用管理工具
