搜索引擎系統

　
 1.平台簡介:
    天盟搜索原名NYS搜索,開發的初衷是協助我們蒐集互聯網上的信息。經過四年的發展，已經成為一套完善的搜索引擎系統，而且是互聯網上第一套公開所有程序（包含部分JAVA代碼）的搜索引擎，它使網管們可以輕鬆搭建真正屬於自己的搜索引擎。 NYS小組早在1998 年就開始搜索引擎研發，現已成功推出基於J2EE Oracle 和ISAPI Oracle兩個版本的成熟的搜索引擎系統，幾乎可適於所有平台。
後台程序運行在網頁採集機器上， 24 小時對互聯網網頁進行超鏈接分析，將數據提取到oracle 數據庫中，採集速度Oracle 每天每台網頁採集機器增加6G 數據。
主服務器的前台利用JSP Oracle 實現基於J2EE的速度最快的搜索網頁。
本系統在Linux 和winNT 測試通過，搜索速度遠高於在目前百度和一搜的PHP的搜索，現在轉讓整套系統，同時也尋找項目合作夥伴和投資商，預算需要資金支持200 萬，主要用於構件高速服務器組，我們堅信憑藉我們不懈努力，一定會成為最大的中文搜索！
本搜索引擎已經在曲阜師範大學校級項目NOAS 等眾多項目中成功應用。
