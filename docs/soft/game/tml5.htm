CIG竞技棋牌游戏平台

　　CIG竞技棋牌游戏平台采用与前几代不同的API开发以及全新的TMLib类库语言重新开发，使用更高性能的Api和原生Intel编译器,在Intel32位处理器上获得更卓越的表现。提供一系列核心技术，包括优化支持，线程应用程序支持，充分发挥超线程技术优势。 CIG竞技棋牌游戏平台是目前游戏界唯一一套用API/SDK开发的网络视频棋牌游戏，无MFC\GDI+\DX。

※CIG竞技棋牌游戏特性
■TMLib技术：底层采用性能更高的TMLib技术，使用WinApi和原生Intel C++ 9.0编译器开发,在Intel32 位处理器上获得更卓越的表现。提供一系列核心技术，包括优化支持，线程应用程序支持，充分发挥超线程技术优势。

■SOA网格架构：客户端采用SOA架构，服务器采用网格架构，理论上能集成一切网络系统。通过这一技术能将游戏的所有运营商和合作商全部集成起来到统一的游戏平台中，最终形成亿万人级别同时在线的大型网络游戏。

■P2P网格技术：将P2P与网格完美结合，不但分布式和异构性大大增强，而且数据传输不需要经过服务器，增大了服务器负载。

■AI人工智能：部分房间可以增加智能机器人陪玩。

■清晰流畅的P2P视频语音：多年研发积累的64路网络视频会议技术实现多人视频游戏。具有完善的丢包检测、智能重发机制、即使64人同桌游戏，也可以同时看到所有人，视频可以全屏放大。



■内置视频压缩器，不需要客户再装H263之类的解码器。码率不超过12K，电话线上网也很清晰。解码加入马赛卡处理、图像圆滑处理……。效果超过QQ超级视频。

■完善的推广系统：整合TMtree.com和在线赠送系统，实现下线抽水、推广赠送、在线赠送等功能。

■更自由界面自定义：游戏大厅采用HTML实现，代码公开，会编网页即可修改游戏界面

■基于浏览器的界面：游戏大厅采用多线程浏览器实现，使用效率更高更简洁的Win32 SDK开发，完全自行封装扁平架构的开发库

■多核处理器支持，支持最新的多核处理器。支持最新处理器的优化功能提高程序在这些处理器上的效能。

■运行时堆栈检查，支持运行时对堆栈的检查，减少缓冲器溢出导致的安全缺陷。

■64位支持，支持IA-32，64位Itanium2以及64位内存。
