FLASH版Web棋牌游戏软件 -- 第二代TML网络游戏FLASH版

1.平台简介

    FLASH版Web棋牌游戏软件是在第二代TML游戏的服务器端的基础上，将原来的客户端换为FLASH开发而成。
开发中，不但吸收了国外众多最新流行的FLASH游戏精华，而且服务器继续沿用TML第二代网络游戏的经典网络结构：由一个中央服务器TMLcenterServer.exe，多个大厅登陆主服务器MainServer.exe和多个房间服务器LocalServer.exe组成集中分布式网络结构。
    另外增加了网络化的辅助管理工具：服务器监控程序ServerMonitor.exe和数据库管理工具DataBaseTool.exe，方便运营和维护，有效降低了管理成本。

2.系统优势

    服务器分中心服务器，登陆服务器，游戏服务器，分布式架构，充分抵挡各种网络攻击，安全可靠稳定。客户端分管理端，游戏端。
管理端功能有自身资料修改，帐号管理（包括存取额度，新建下线帐号与子帐号，基本资料修改，详细资料修改，权限修改），游戏报表查看（两种查看方式），游戏记录查看（包括时间，桌号，出牌情况，占成洗码等），现场投注（包括会员投注情况，踢人等）操作记录查看（包括密码修改，占成比修改，洗码比修改，权限修改等）游戏端功能有密码修改，游戏报表查看，游戏记录查看，大厅现场直播，详细游戏帮助等。
    前端客户端采用最新Flash9技术，通过直接原始Socket与服务器通信，不存在.Net,FMS等中间层，更加高效，稳定，安全。后台服务器采用Visual C++ 2003工具开发,更加高效，可靠，稳定
