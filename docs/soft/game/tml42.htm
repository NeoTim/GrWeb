   ■第四代视频语音棋牌游戏系统(第二版)

第四代服务端实现了推广系统，而第五代客户端实现了自由成绩单的控制。
两者使用不同开发语言和方式开发。
因客户要求两个功能要在同一平台下实现，所以将第四代的服务器端和第五代的客户端，通过公共的TCP和UDP协议结合起来，形成第四代(第二版)。

第四代(第二版)在客户的不断要求和不断修改下，它的功能已经和我们的本意不符了，它的客户端和服务器端是两种不同的技术实现。
服务器端与第四代完全一致，而客户端与第五代几乎完全相同。
