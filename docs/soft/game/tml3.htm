炼金老虎机、象棋、围棋、QQ斗地主、2人暗牌斗地主、普通斗地主、斗地主两副、疯狂斗地主、连线斗地主、重庆斗地主、皇嘉斗地主、升级、三打一、跑得快、普通四冲、沈阳四冲、原子、双扣、百变双扣、510K、憋七、玉环80分、玉环打通、嵊州105、红心大战、拱猪、锄大地、炒地皮、红十、四国军棋、联众麻将、温岭麻将16张、上海麻将、大庆麻将、湖北麻将、疯狂麻将13张、梭哈、保皇、21点、赢三张(扎金花)…………

2.面对面视频棋牌游戏技术简介

   视频版棋牌游戏采用h.263压缩视频，所有40款游戏均带视频功能。视频版棋牌游戏由游戏服务器、客户端大厅、游戏客户端DLL构成。每个游戏均为一个独立的DLL文件，方便分组开发和外包开发。 游戏大厅服务器采用TCP短连接模式，单台服务器可承载5000以上用户同时使用。 游戏服务器采用TCP长连接，单台游戏服务器可同时支持5000人在线游戏，如进行某些优化可实现更多用户支持。视频版棋牌游戏服务器单独记录游戏用户积分状态，并采用完全可分布、可维护的摆放方案。可动态扩展服务器，定制游戏房间、游戏桌。 服务器充分利用NT系统能力，底层结构清晰，并可方便的开发新游戏、新游戏服务器。 客户端采用GDI+模式工作，对于棋牌游戏可充分支持，如需进行复杂图像操作，可套用DX\DDRAW进行扩展，实现靓丽的游戏表现。
客户端开发模式简单，接口清晰。对于一般类型的棋牌游戏，可在1周内实现一款。
服务器代码实现所有的网络、数据库、棋牌桌管理的功能。
客户端包含一个用于开发游戏所需要的DLL框架工程。即使生手也可以在短时间内进入开发状态。
安全性：
数据库部分敏感信息加密、客户端IPC通信和网络通信都采用自主开发的加密技术、客户端用户密码加密等，同时，对网络包采用自我的加密算法加密，本地保存的敏感信息采用多层加密，有可逆的与不可逆的，数据库部分的敏感信息采用 MD5算法加密，还利用端口变换，服务器部署环境增加安全和保密；保证不会由于系统、游戏漏洞造成游戏金币流失，外挂等状况。

3.面对面视频棋牌游戏优势

高压性：
单台服务器最高并发可支持5000人同时在线。人数超过5000时，可进行多台服务器分布式处理，网络、CPU占用率极低，加强了平台对服务器要求的伸缩性。
* 1. 每游戏服务器的安全并发用户人数达到4000人以上。并发用户人数的上限达到5000人以上。服务器配置为XEON 2.4G ×2、1024M×2内存、73G硬盘。
* 2. 当平台人数达到4000人时，在带宽512K的ADSL的条件下，用户登录、进入游戏房间等的平均时间不高于20秒。
* 3. 每聊天室的安全并发用户数达到500人。
* 4. 当人数达到500人时，聊天信息从发送到显示的时间不高于5秒。网络环境为512K ADSL。
* 5. 正常维护频率不高于2个月维护一次，不需要停机维护，每次维护在半小时以内。

4.面对面视频棋牌游戏系统架构

    本系统可实现完全分布的部署外，同时也支持分布式开发，只要将必要的.h、.lib、.dll提供出来（有SDK生成工具），即可实现技术隐藏的分布式开发，有助于提供更多开发者同时工作，并保护内部技术细节。
游戏登陆：
在登陆服务器选项中可以用多个域名或者IP登陆，支持建立多个分站，为合作运营提供强大支持。
游戏房间：
* 1、 视频版棋牌游戏的同一游戏可开多个游戏房间，房间数可无限增加，并可针对不同级别的玩家设置开启不同级别的游戏房间，满足不同用户需求。如：开设会员房间、比赛房间、练习房间、限制最低积分或金币的游戏房间等等
* 2、 每个视频版棋牌游戏房间在服务端可设置最高上限人数
* 3、 视频版棋牌游戏积分房间和金币房间共同存在，每款游积分有单独数据库统计，金币房间统一数据库。
* 4、 用户可同时进入多个积分游戏房间而不能进入多戏的个金币游戏房间，使金币游戏的安全性提高。
