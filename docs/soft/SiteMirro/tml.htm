 b.平台简介
    SiteMirro是国产第一款网站镜像服务器。它基于NT后台服务的形式，提供镜像网站服务，它能将一台服务器变为多个网站的镜像网站，理论上可以镜像六万多个站点。本软件也是国内使用最广泛的网站镜像软件。1999年至今曾担任过中国学术期刊、书生之家15万册全文电子图书、重庆维普外文期刊文摘、万方数据等镜像服务。SiteMirro的新版本已不是简单的镜像软件，它最独特优点是能够自动生成网站镜像，您只要设定好母站，它会在本机完全原样生成网站镜像，而且它自带web服务器，可直接使用镜像服务，它的web服务器完全原样显示ASP、PHP等所有脚本的运行情况，因为它制作网站镜像时候，已经将这些映射为HTML了！完全不用我们做任何事情，只要我们选择的分服务器上有镜像大师，就能建立我们的镜像网站，SiteMirro操作简单、功能强大、速度稳定，是您首选的镜像工具。
 b.企业版功能简介
   考虑到技术保密，这里简单介绍两种方案。完善方案只给最终用户提供。
方案一：同域名镜像是一种能为企业站点提供更快速、更稳定、更安全服务的网络服务。在不同地区访问同一域名的时候，会自动转到本地区的服务器。如yisou,yahoo,baidu等站点均采用类似的机制。
转向功能由SiteMirro提供。 其技术原理是采取分布式网络缓存结构（即国际上先进的 web cache 技术），通过在现有的Internet中增加一层新的网络架构，根据来访用户IP地址，智慧分配用户的请求，并将网站的内容传送到最接近用户的cache服务器内，让来访用户从就近cache服务器内读取所需数据，全心为用户营造一个畅通无阻的网络环境，可以助您的网站访问流畅，企业的电子商务更成功！。
SiteMirro为您提供多个镜象站点，根据不同地区改变访问路径，保证快速访问。对于电子商务类网站，有所谓8秒定律，即用户访问一个网站时，如果等待网页打开的时间超过8秒，会有超过30%的用户放弃等待。据资料显示，估计每年由于网络问题 (缓慢甚至不通)导致电子商务方面的损失高达44亿美元。
SiteMirro切换过程简单易行，操作方便。 网络覆盖全：保证不同地区和不同网络中的用户都能得到良好的访问质量。 完善冗余机制：保证在设备或节点出现故障时，都不会影响用户的正常访问。 强大安全保障机制：有效地预防黑客入侵以及降低各种D.D.o.S攻击对网站的影响。 成功案例：某网站开始使用我们的服务时，只用了短短五个月访问数量增加了6-7倍多，而访问质量不但 没有下降反而得到提高，5月前访问响应速度10-20秒，使用后则在8秒以内，而这期间设备没有增加， 服务质量得到提高，成本得到有效控制，交易额也获得迅速增长，起到了一举多得的效果。
方案二：同一网站的多点镜像。在不同网络(如电信、网通、铁通、联通等)建立不同的镜像服务器，镜像服务器存放相同的内容。同步问题由我们的软件解决。如某些网站的“安徽长城宽带镜像 苏州热线镜像 四川自贡日报镜像”等。
SiteMirro具备为自动镜像、触发镜像、缓存镜像三个功能。
自动镜像是将主服务器的内容定时更新到所有镜像服务器。适合于一般站点。
缓存镜像可以根据访问者的的请求，如搜索等结果保存到镜像服务器。适合于海量数据库服务器。
触发镜像指在SiteMirro主服务器更新某些文件时，能够触发SiteMirro文件监视服务，将更新的文件，自动部署到各个SiteMirro镜像服务器。适合大型文件，如软件及电影的站点。
个性化配置：自动镜像有年、月、日，文件类型过滤、文件规则等多种参数。触发镜像和缓存镜像也有相同规则的参数。由于技术保密，因此具体功能只向最终用户介绍。通过合理的设置自动镜像、触发镜像、缓存镜像三个功能的参数，可以搭配出最适合您的站点的镜像方案。 目前我们一直在后台为中国数十家大型网站提供提供镜像服务。
