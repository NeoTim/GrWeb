基于OCI开发的ORACLE独立ADO驱动　集成超级管理器—告别SQL*Net的真正高速直连

　
 您现在所在的位置:Oracle++ DLL驱动(包含Oracle Provider for OLE DB)
—使VisulFoxPro、FLASH、Authorware、Director高速直连Oracle
 1.下载试用
本驱动是基于OCI封装的一组Oracle函数的动态库。使开发者可以在VisulFoxPro、FLASH、Authorware、Director等 不支持Oracle存储过程的开发工具中，直接开发高速稳定的Oracle应用程序。免费测试版下载地址：http://tml.cn/soft/OCI/dll.rar（压缩包中包含了一套VisulFoxPro连接Oracle源代码，和一套HTML连接Oracle的源代码）
 2.VisulFoxPro开发Oracle应用程序说明>
　　Oracle与Visual FoxPro的数据引擎是毫无关系的系统，Visual FoxPro对Oracle的支持只是停留在较低的水平。Oracle的Recordset无法与Visual FoxPro的内置控件捆绑。其原因是：Visual FoxPro 不能识别Oracle的Recordset，Visual FoxPro只能通过COM的方式识别Oracle的Recordset，不能像表格那样读写它，只能将记录逐条读取，而不是一下子认得整个Recordset。这样Recordset无法与Visual FoxPro的内置控件捆绑，同时也不能对Oracle直接使用Visual FoxPro强大的数据处理功能。
　　Oracle++ DLL驱动可以使Visual FoxPro高效读写，在Oracle6i以上版本和Visual FoxPro6以下版本中测试通过。在我们提供的测试下载中包含一套VisulFoxPro连接Oracle源代码。调用方法： //格式为：d=TML.conn("ORACLE服务器IP,ORACLE服务器端口,ORACLE服务名,ORACLE用户名,ORACLE用户密码");
 3.开发说明
使用本驱动开发Ｏｒａｃｌｅ程序的步骤：
1.安装本驱动
2.在程序中创建对象
如:在FOX中使用：TML=CREATEOBJECT("TML.TMLi")
在JAVA中使用：TML=new ActiveXObject("TML.TMLi");
在VB中使用：set TML=CreateObject("TML.TMLi")
在Delphi中使用：users ComObj var TML:Variant; TML:=CreateOleObject('TML.TMLi');
这里创建对象名为TML.TMLi,其中库名为TML,其中TMLi为对象 该DLL中封装了我们独立开发的ORACLE驱动，以及SQL解释器,这是一组函数库，可以在JAVA里调用，也可以在FOXpro等，支持DLL函数库的都可以用。这句话是首先要创建函数库，这个函数库中对象名就是TML.TMLi，这个是固定的，在FOX也是这样，就象ZIP
FOX操作Oracle时，不需要AOD，也不需要ODBC，等，不需要其他任何东西，只需要安装OCIdriverDLL.exe就行了。
3.连接打开Oracle
FOX和VB中使用d=TML.conn("ORACLE_HOST_IP,ORACLE_PORT,ORACLE_SID,ORACLE_User,ORACLE_Password")
JVAA中使用：d=TML.conn("ORACLE_HOST_IP,ORACLE_PORT,ORACLE_SID,ORACLE_User,ORACLE_Password");
上面的d为返回的字符，会提示是否连接成功。.conn是这个函数库的一个方法 该方法用于设置oracle的连接参数并连接Oracle。
参数是ORACLE_HOST_IP,ORACLE_PORT,ORACLE_SID,ORACLE_User,ORACLE_Password
分别是ORACLE服务器IP，服务器端口，数据库ID（如LOCALDB），数据库的用户名，和密码。
中间用,隔开。在第一次打开ORACLE服务器只前，首先要调用一次，
如果程序中，需要更改ORACLE服务器，换个用户或换个数据库，再要调用一次，更新连接，
如果程序一直用同一个库，同一个用户，那不需要再调用。.conn会返回一个字符串，如果连接成功就返回连接信息，如果连接失败就连接失败的错误代码。
4.操作Oracle（Insert,Delete,CREATE TABLE,creat user,creat index，creat function，creat ……等）
FOX和VB中使用：d=TML.ExeSql("insert into T(TEST) values('FOX操作ORACLE BY FOXPRO')")&&;//插入一条记录
JAVA或JS中使用：d=TML.ExeSql("insert into T(TEST) values('JavaScript操作ORACLE')");//插入一条记录
上面的d为返回的字符，会提示操作的信息。.ExeSql是执行SQL语句的方法，参数是可以是任意DML语句（如insert,delete,update），也可以是任意DDL语句
如（creat table,creat user,creat index，creat function，creat ……等，）也可以是Drop等。
Oracle所有的SQL语句，都支持，但不返回数据集。但返回一个结果，如update语句更新的数据库记录数。
上面的语句"insert 就是向T表中的插入一条记录
T表定义如下：CREATE TABLE "T" ("ID" NUMBER(10) NULL,"TEST" VARCHAR2(50) NOT NULL)
可以用 TML.ExeSql("CREATE TABLE T (ID NUMBER(10) NULL,TEST VARCHAR2(50) NOT NULL)")来创建这个表。5:查询Oracle（select）
FOX和VB中使用：s=TML.SelSql("Select id,TEST from T where test like '%%' order by id desc",25,document.cookie)
JAVA 或JS中使用：s=TML.SelSql("Select id,TEST from T where test like '%%' order by id desc",25,document.cookie);
SelSql是查询的方法，专门用来执行SELECT语句。输入三个参数的格式：SQL语句必须是SELECT,每页行数,页数
返回为一个S的字符串，S是一个二维数组，需要自己还原成一个表格，，返回的字符串用“||”来分割每条记录，用“|”来分割字段
也就是说两个“||”间的是一条记录，两个“|”间是一个字段，字段顺序就按SELECT中的顺序来排列的，所以SELECT中必须指明字段名，不能用*，但可以用MAX，SUM
//查询数据库的语句，格式：SQL,页行数,页数。
