■“域之光”域名掃描器

簡介：

“域之光”域名掃描器是由國軟推出的國內第一套域名掃描器。國軟是國內優秀的域名註冊商之一。為您提供最好的域名註冊服務。

最新版下載:http://tml.cn/soft/domscan/domscan.exe

功能：
　1、迅速幫你找到固定前綴或後綴的好域名，比如hk163.com：海口信息港就是通過此軟件快速註冊的，只要固定前綴hk，所有前兩個字母為hk的未註冊的域名將立即列出，您只要選擇最喜歡的，然後到nic.tml.cn自行註冊即可
2、迅速幫您找到位數最少的域名。位數越少，價值越高，本軟件可以幫你在網上迅速淘金。
3、迅速找到固定字母範圍的域名，可以在您指定的字母範圍內完整的歷遍所有組合。第一時間找到位數最少的未註冊域名。
4、支持字典掃描，迅速找到純單詞的完美域名

升級歷史：

V1.0 可按字典進行掃描
V2.0 增加歷遍功能,可以按固定位數自動組合進行掃描。
V2.2 增加可按照固定前綴或後綴掃描功能
V3.0 增加自動多位掃描功能
V3.2 增加多服務器設置
