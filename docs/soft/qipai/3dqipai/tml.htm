五、     用辅助管理工具修改机器人或者用户游戏币：

①     进入辅助管理工具；

找到admin.tml 并打开！

②     打开其他辅助工具集（admin.tml）：

③     输入以下代码

1、 给所有机器人加游戏币：

Update User_1 set point=point+XXXXX where UserID in (select UserID from Users where Member=255)

注：XXXXX是机器人要加游戏币数目；

2、给所有用户加游戏币：

Update User_1 set point=point+XXXXX where UserID in (select UserID from Users where Member<>255)

3、如果要给所有人（包括机器人）加游戏币：

Update User_1 set point=point+XXXXX

不需要后面的条件。红色是要输入的代码；

注：XXXXX是机器人要加游戏币数目；以上代码如果输错则执行不了。

4、其他快捷语句请慎重使用，最好在详细询问客服人员后，再操作使用，如有误操作，本公司概不负责。
